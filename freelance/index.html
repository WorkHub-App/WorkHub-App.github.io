ПИТАННЯ

Часті запитання

Чи потрібен ФОП для роботи?

Ні. WorkHub можна використовувати як фізособа. Для оплати підписки — картка, Monobank або криптовалюта.

Чи можна виставити рахунок без реєстрації бізнесу?

Так, достатньо заповнити свій профіль (ім'я, контакти, за бажанням — реквізити) і рахунок згенерується в PDF з вашими даними.

Чи можна приймати оплату від іноземних клієнтів?

Рахунки підтримують оплату криптовалютою (USDT, BTC, ETH), що зручно для міжнародних клієнтів без банківського переказу.

Чи веде застосунок облік часу по проектах?

Так, вбудований таймер прив'язується до конкретного проекту чи задачі.

Де зберігаються дані?

У Firebase — вашій приватній хмарі. Застосунок працює і офлайн, синхронізується при підключенні до інтернету.

Як скасувати підписку?

В будь-який момент через Налаштування → Підписка. Акаунт переходить на FREE план, дані не видаляються.

Готові прибрати хаос у роботі?

Встановлення займає менше хвилини. Безкоштовно, без картки.
